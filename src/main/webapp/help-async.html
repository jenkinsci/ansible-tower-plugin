If checked, the job will be launched in AAP and will then return immedatly. You can capture the object returned like this: aap_job = Things you might want to do with this object: aap_job.isComplete() - returns true of false or throws an AnsibleAAPException aap_job.getLogs() - returns a Vector<String> of all the logs or throws an AnsibleAAPException aap_job.getExports() - returns a HashMap<String, String> or throws an AnsibleAAPException Note: you should not call getExports until getLogs is run
